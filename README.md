# React+Ts Template

Template basico de React con Typescript

## Tecnologias

1. React
2. Typescript
3. Eslint
4. Prettier

## Instalacion

1. Clone el proyecto
2. Ejecute `npm install`
3. Ejecute `npm run dev`

## Comandos extras

```bash
# Busca y corrige errores en el codigo
$ npm run lint

# Formatea el codigo
$ npm run format
```
